The commit SHA above is the frozen source: architecture, training code, and both
selected weights (Git LFS). It was pushed before the private images were released,
and nothing affecting the model has changed since. Only inference, CSV merging, and
documentation changed after release, as the rules allow.

Method: a no-TTA weighted rank ensemble of a ConvNeXt-V2 Base detector (weight 0.75,
512 px) and a DINOv2 ViT-B/14-with-registers detector (weight 0.25, 392 px). Each
model emits a sigmoid fraud score; scores are converted to average ranks over the
mounted test set and combined. Higher means more likely fraudulent.

Reproduction:

  git clone https://github.com/Marc-Dvci/FREUID && cd FREUID
  git checkout 4db0ae5fd62c739a9398175c78e32add4f418a22
  git lfs pull
  docker build --pull -t freuid-repro:local .
  docker run --rm --gpus all --network none --read-only \
    -v /path/to/flat/test/images:/data:ro -v "$(pwd)/out:/submissions" \
    freuid-repro:local

This writes exactly /submissions/submission.csv with columns id,label, one row per
image, the id being the filename stem. Scoring the 134,997 private images took 131
minutes on a single RTX 4070.
